关于 HPL

HPL（H Programming Language）是一种基于 YAML 格式的面向对象编程语言，使用动态类型。

  • 📝 YAML 格式语法
  • 🏗️ 面向对象（类、继承、对象）
  • 🔄 控制流（if/else、for、while）
  • 📦 标准库模块（math、io、json、os、time）
  • ⚡ 内置函数（echo、len、type 等）
